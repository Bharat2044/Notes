				=================================================
							5. str()
				=================================================
=>str() is used for converting All types of values into str type.
=>Syntax:        varname=str(int / float / bool / complex )
-------------------------------------------------------------------------------------------------------------------------------------
Examples
-------------------------------------------------------------------------------------------------------------------------------------
>>> a=123
>>> print(a,type(a))
123 <class 'int'>
>>> b=str(a)
>>> print(b,type(b))
123 <class 'str'>
>>> b
'123'
---------------------------------
>>> a=12.34
>>> print(a,type(a))
12.34 <class 'float'>
>>> b=str(a)
>>> print(b,type(b))
12.34 <class 'str'>
>>> b
'12.34'
-----------------------------------
>>> a=True
>>> print(a,type(a))
True <class 'bool'>
>>> b=str(a)
>>> print(b,type(b))
True <class 'str'>
>>> b
'True'
-----------------------------------------
>>> a=2+3.5j
>>> print(a,type(a))
(2+3.5j) <class 'complex'>
>>> b=str(a)
>>> print(b,type(b))
(2+3.5j) <class 'str'>
>>> b
'(2+3.5j)'
